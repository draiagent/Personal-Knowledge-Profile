# Personal Knowledge Profile

> Engineering × AI × Business × Healthcare × Education

陳俊宏（Chen, Chun-Hung）的人物知識核心（Personal Knowledge Profile, PKP）。

本儲存庫作為個人資料的唯一可信來源（Single Source of Truth），供 GitHub、Obsidian、履歷、官網、演講簡介、出版、自媒體與 AI 協作共同使用。

## 核心入口

- [PROFILE.md](PROFILE.md)：完整自我介紹
- [INDEX.md](INDEX.md)：人物知識索引
- [TIMELINE.md](TIMELINE.md)：重要時間軸
- [education/](education/)：學歷、論文與持續教育
- [career/](career/)：企業經營、職涯與組織服務
- [teaching/](teaching/)：大學授課、企業培訓與演講
- [research/](research/)：研究方向與出版成果
- [professional/](professional/)：證照、聘書、獎項與感謝狀
- [content/](content/)：YouTube 與媒體內容

## 維護原則

1. 所有資訊以已確認資料為準。
2. 一個檔案只負責一類資訊。
3. 新增學歷、證照、授課、演講、論文或成果時，更新對應檔案。
4. `PROFILE.md` 僅保留人物總覽與核心定位。
5. 原始證明文件規劃集中於 `assets/`（尚未建立，待掃描檔到位後補上）。
